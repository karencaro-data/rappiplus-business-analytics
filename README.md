# 📊 RappiPlus Business Analytics

## 📌 Descripción del Proyecto

Este proyecto presenta un diagnóstico estratégico integral para **RappiPlus**, una empresa de comercio electrónico, mediante el análisis de datos para identificar oportunidades de crecimiento, optimizar el desempeño comercial y apoyar la toma de decisiones basada en datos.

El análisis incluye limpieza de datos, construcción de indicadores de negocio (KPIs), análisis de ventas, evaluación del gasto en marketing, análisis de cohortes, funnel de conversión, experimento A/B y desarrollo de dashboards interactivos en Tableau.

# 📈 Resultados Clave

Durante el análisis se obtuvieron los siguientes indicadores de negocio:

| KPI | Valor |
|------|-------:|
| 💰 Revenue Total | **44,259,659** |
| 📈 Profit Total | **7,551,224** |
| 📣 Marketing Spend | **2,694,664** |
| 🛒 Ticket Promedio | **2,074.80** |
| 📦 Productos promedio por orden | **7.06** |

### Hallazgos principales

- ✅ La categoría **Electrónica** concentra la mayor parte de los ingresos.
- ✅ **Laptop-Gaming-16GB** fue el producto con mayor rentabilidad y volumen de ventas.
- ✅ El análisis de cohortes permitió identificar oportunidades para mejorar la retención de clientes.
- ✅ El análisis del embudo (Funnel) ayudó a detectar oportunidades de optimización en el proceso de compra.
- ✅ El experimento A/B proporcionó evidencia estadística para apoyar decisiones sobre la experiencia del usuario.

# 🎯 Objetivo del negocio

Desarrollar un análisis integral que permita:

- Evaluar el desempeño comercial.
- Medir la rentabilidad del negocio.
- Analizar la efectividad del marketing.
- Comprender el comportamiento de los clientes.
- Detectar oportunidades de mejora mediante análisis de cohortes y funnel.
- Validar cambios mediante un experimento A/B.

# 🧹 Limpieza y preparación de datos

Se realizaron las siguientes actividades:

- Validación de tipos de datos.
- Conversión de fechas.
- Eliminación de registros duplicados.
- Verificación de valores nulos.
- Estandarización de variables categóricas.
- Creación de variables calculadas para análisis posteriores.

Estos indicadores permiten evaluar el rendimiento financiero y operativo de la empresa.

# 📈 Análisis realizado

## Dashboard Ejecutivo

Incluye:

- KPIs principales
- Revenue mensual
- Revenue por categoría

Permite monitorear rápidamente el desempeño general del negocio.

---

## Dashboard de Detalle

Incluye:

- Profit por producto
- Cantidad vendida por producto
- Tabla detallada de órdenes
- Filtros por:
  - País
  - Categoría
  - Fecha

Este dashboard facilita el análisis detallado para la toma de decisiones.

---

## Funnel de Conversión

Se construyó un embudo de conversión para medir el comportamiento de los usuarios durante el proceso de compra.

El análisis permitió identificar oportunidades para mejorar la conversión entre etapas.

---

## Cohortes de Retención

Se desarrolló un análisis de cohortes para evaluar la retención de clientes a lo largo del tiempo.

Esto permitió identificar patrones de permanencia y abandono de usuarios.

---

## Experimento A/B

Se analizó un experimento A/B comparando dos grupos de usuarios.

El objetivo fue determinar si existían diferencias estadísticamente significativas en la conversión entre ambas versiones.

# 💡 Principales hallazgos

- El Revenue presenta variaciones importantes entre meses.
- La categoría Electrónica concentra la mayor parte de las ventas.
- Algunos productos generan una rentabilidad considerablemente mayor que otros.
- El análisis de cohortes permitió identificar patrones de retención.
- El funnel evidenció oportunidades para mejorar la conversión.
- El experimento A/B permitió evaluar el impacto de cambios sobre el comportamiento de los usuarios.

---

# 🚀 Recomendaciones

- Optimizar las campañas de marketing hacia productos con mayor rentabilidad.
- Implementar estrategias para aumentar la retención de clientes.
- Mejorar las etapas del funnel con menor conversión.
- Priorizar las categorías con mayor margen de beneficio.
- Continuar utilizando experimentos A/B para validar decisiones comerciales.
